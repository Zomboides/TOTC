========================================
TOTC: Think Of The Children
Installation Guide
========================================

QUICK INSTALL:

1. Copy the folder "Contents/mods/TOTC_ThinkOfTheChildren" to:
   Windows: %USERPROFILE%\Zomboid\mods\
   Linux: ~/.local/share/Zomboid/mods/
   Mac: ~/Library/Application Support/Zomboid/mods/

   Result: Zomboid/mods/TOTC_ThinkOfTheChildren/
           ├── common/
           └── 42/

2. Launch Project Zomboid

3. Go to Mods menu → Enable "TOTC: Think Of The Children"

4. Start a new game or load existing save

5. Configure in Sandbox Options > TOTC tab (optional)

========================================
SETTINGS:
========================================

Child Zombie Spawn Chance (0-100%)
  Default: 10%
  Controls percentage of zombies that spawn as children

Child Zombie Size (10-100%)
  Default: 50%
  Controls size relative to normal zombies

========================================
COMPATIBILITY:
========================================

✓ Project Zomboid Build 42.13.1+
✓ Singleplayer and Multiplayer
✓ Most other mods

NOTE: Uses Build 42 folder structure with
common/ and 42/ version folders.

========================================
TROUBLESHOOTING:
========================================

Mod not showing:
- Verify folder structure (Build 42):
  TOTC_ThinkOfTheChildren/
  ├── common/           <- REQUIRED (even if empty)
  │   └── media/
  └── 42/
      ├── mod.info
      ├── poster.png
      └── media/
          ├── sandbox-options.txt
          └── lua/...
- Check mod is enabled in Mods menu
- Review console.txt for errors

Settings not working:
- Start a NEW game after enabling
- Check Sandbox Options > TOTC tab

Multiplayer:
- Host AND clients must have mod enabled
- Settings controlled by server
